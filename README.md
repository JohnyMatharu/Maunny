# Maunny

# Description
A very common problem that people are facing now a days when dealing with when purchasing a vehicle is that they have cannot get comprehensive information that would assit them in making a correct purchase decision. This is why they are stuck with a wrong vehicle for years, and something that even does not function properly. Maunny is a web application which acts as an online dealership, a basic car marketplace where users can search a vehicle from available inventory, calculate payments and submit their interest. 

The app currently is organized in three sections which is vehicle search, customer information and finance payment calculator. The layout was created with HTML, with additional styling via custom CSS. JQuery was used in combination with javaScript. Vehicle search is backed by inventory database which will be updated with latest inventory from dealers. Finance paymant calculator is an estimate which uses the most accurate formulaes and is based on credit scores which is one step away from credit consult online with finance manager. Customer information is stored and customer will be contacted by email for more details. 

User can run this app starting with first section, i.e. search the name of a car manufacturer, car model, features including mileage, and year of manufacturing. While using the payment calculator, they can submit down payment/exchange car, interest rate with their credit scores, term, and choice of payment interval. User can choose the stock number and submit it at customer information section to show interest and get response, data will be stored in the local storage and customer can see on their machine for their own record history.
